CONVERSOR TOTVS - GUIA DE INSTALAÇÃO

=======================================

BEM-VINDO AO CONVERSOR TOTVS!

Este instalador guiará você através da instalação do software.

REQUISITOS DO SISTEMA:
- Windows 10 ou superior
- 50 MB de espaço em disco
- 4 GB de RAM recomendados

INSTRUÇÕES DE INSTALAÇÃO:
1. Execute este instalador como Administrador
2. Siga as instruções na tela
3. Escolha o diretório de instalação (recomendado)
4. Aguarde a conclusão da instalação

APÓS A INSTALAÇÃO:
- Um atalho será criado no Menu Iniciar
- Um atalho será criado na Área de Trabalho
- Para desinstalar, use "Adicionar ou Remover Programas"
